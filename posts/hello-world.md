# 你好，世界 👋

这是我的第一篇博客文章，也是在这个站点上留下的第一行正式内容。

## 为什么要写博客？

- 记录自己的想法和成长
- 把零散的笔记整理成系统的文章
- 未来回头看的时候，能看到自己走过的路

## 这个博客是怎么搭建的？

- **技术栈**：纯前端（HTML + CSS + JavaScript）
- **文章格式**：全部使用 Markdown 来书写
- **部署方式**：可以直接托管在 GitHub Pages 上

## 我打算写些什么？

- 前端技术与实践
- 学习笔记与踩坑记录
- 一些不那么严肃的生活随笔

## 最后

如果你能看到这里，说明这个博客已经基本成型啦。

接下来要做的事情，就是 —— **持续写下去**。
