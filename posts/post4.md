# 重新开始写博客的这一天

日期：2026-04-09

今天正式把这个小博客从“纯静态网页”升级成了一个真正的博客系统，可以：

- 用 Markdown 写文章
- 自动生成文章列表和详情页
- 部署到 GitHub Pages，对外公开访问

## 从静态页面到博客系统

一开始，这个网站只是一个单独的 `index.html`，里面写死了几段文字，想发新文章就要自己去改 HTML，非常麻烦。

后来我做了这些事情：

1. 设计了首页和文章详情页（`index.html` + `post.html`）
2. 把每篇文章都改成用 Markdown 存在 `posts/` 目录下
3. 用 JavaScript 在前端读取 `posts/posts.json`，自动渲染文章列表
4. 用 `marked.js` 在浏览器里把 Markdown 转成 HTML

现在，只要写 `.md` 文件，就能有一篇新的博客文章出现，体验比以前好太多了。

## 写作这件事

搭好博客只是开始，更重要的是：

- 习惯把零散的想法整理成文字
- 把做过的事情、踩过的坑，记录下来
- 未来回头看的时候，可以看到自己的变化

我不要求自己每一篇都写得多“深刻”，更重要的是**持续输出**，哪怕是一点点学习记录、生活碎片，也值得被好好存放在这里。

## 接下来想写什么？

先给自己列几个方向：

- 前端相关的小知识和实践
- 学习笔记：看书 / 上课 / 看视频的收获
- 日常生活的随笔，比如今天发生的小事、一些小想法

这篇文章就当成是一个新的起点，之后我会慢慢把这个博客填满。
